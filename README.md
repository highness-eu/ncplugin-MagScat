# The MagScat plugin for NCrystal

This repository contains code for the NCrystal plugin named `MagScat`.

This plugin provides paramagnetic scattering calculations for O₂-hydrate and O₂-C₆₀, based on the formulas developed by O. Zimmer (https://doi.org/10.1103/PhysRevC.93.035503). To credit this plugin, please cite Xu et al. (https://doi.org/10.1088/1361-648X/ad5947).
